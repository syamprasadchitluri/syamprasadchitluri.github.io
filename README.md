# Personal Website

This project is deployed at https://syamprasadchitluri.github.io

## About the Project

This is a personal website that showcases my skills, and project. The website is designed to give an overview of my work and provide easy access to my contact information.

The website is built using HTML, CSS, Bootstrap and Javascript. It includes multiple pages and responsive design to ensure an optimal viewing experience on different devices.

## How to Run the Code

To run the code, follow these steps:

1. Clone this repository to your local machine.
2. Navigate to the project directory in your terminal.
3. Run `index.html` in your preferred web browser.
   - Alternatively, you can use the Live Server extension in Visual Studio Code to run the website.
